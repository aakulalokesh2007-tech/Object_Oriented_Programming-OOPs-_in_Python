## Area Calculator
==================================

This repository contains a simple Python script demonstrating the fundamentals of Object-Oriented Programming (OOP). It features a basic class structure used to calculate the area of a rectangle.

## Description
-----------
The script defines a class named `vol` which takes two parameters (length and breadth) upon instantiation. It includes a method to calculate and return the area based on these parameters. 

Note: This script demonstrates that Python handles instance method parameters positionally by creatively using `me` instead of the standard `self` convention.


---------------
## Prerequisites:
- Python 3.x installed on your machine.


```
How to Run:
1. Clone this repository or download the source code file.
2. Open your terminal or command prompt.
3. Navigate to the directory containing the script.
4. Run the script using Python:
   python __init__.py

```
